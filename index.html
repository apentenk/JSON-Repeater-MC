JSON Repeater MC

JSON Repeater MC is a tool that allows you to quickly replace multiple marked keywords in text files at the same time. Perfect for use in editing multiple similar JSON files when modding minecraft. You can choose from the saved templates below, to generate multiple related files. Alternatively you can upload a JSON file direct or type directly into the text box.

HINTS

Keywords are marked by enclosing them in /**/ like /*so*/

If you try to replace a keyword with /**/ it will simply remove the delimiters
